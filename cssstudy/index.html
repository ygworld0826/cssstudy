1.0 기초 Javascript 에 대하여

Javascript는 웹 브라우저에서 동작하는 인터랙티브한 기능을 구현하는 프로그래밍 언어입니다.

Javascript 의 주요 개념

  • 변수:
    • var, let, const를 사용해 데이터를 저장하고 관리
  • 데이터타입: number, string,boolean,object,array, null, undefined, symbol 등
  • 함수: 코드 블록을 정희하여 재사용할수 있으며, function 키워드나 ()=>{}로 선언
  • 스코프: 변수의 접근 범위를 결정하며,global, function,block 있음
  • 이벤트: 클릭,입력 등의 사용자 동작을 감지 addEventlistener를 활용해 처리가능
  • 비동기처리: setTimeout, Promise, async/await 등을 사용해 순차적 실행이 아닌 놀니적 흐름 제어
  • dom조작: document.querySelector(), innerHTML 등을 이용해 HTML요소를 동적으로 변경
  • 객체 지향 프로그래밍:class 문법과 prototype을 활용해 객체 기반 프로그래밍을 구현
  • 모듈화: import와 export를 사용해 코드를 파일 단위로 분리하여 관리
  • 에러처리: try...catch문을 사용해 코드 실행중 발생하는 오류 제언

1.0 CSS에 대하여

CSS(Cascading Style Sheets)는 웹 페이지의 시각적 스타일을 정의하는 스타일 시트 언어입니다.

CSS의 주요 개념

  • 선택자 (Selector) - 스타일을 적용할 HTML 요소 선택
  • 속성 (Property)과 값 (Value) - 스타일을 설정할 속성과 값 지정
  • 박스 모델 (Box Model) - HTML 요소의 레이아웃 구성
  • 플렉스박스 (Flexbox) - 요소를 가로/세로 정렬
  • 그리드 (Grid) - 2차원 배치 시스템
  • 미디어 쿼리 (Media Queries) - 반응형 웹 디자인

이전 블로그 내용 요약

CSS(Cascading Style Sheets)는 HTML 요소의 디자인과 레이아웃을 정의하는 스타일링 언어로, 색상, 크기, 배치 등을 조정하여 웹페이지의 시각적 표현을 담당합니다.
